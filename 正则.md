##### 1.匹配指定字符串开头的银行卡号

```
/^621623001\d{7,10}$/
```

##### 2.字母和数字

```
/^[A-Za-z0-9]+$/
```

##### 3.手机号

```
/^1(\d{10}|\d{12})$/

/^[1][34578]\d{9}$/
```

##### 4.座机

```
/^(0[0-9]{2,3}\-)?([2-9][0-9]{6,7})+(\-[0-9]{1,4})?$/
```

##### 5.IdCard

```
/\d{15}|\d{18}/
```

##### 6.邮箱

```
/\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*/
```

##### 7.校验带小数点的

```
/^\d+(?=\.{0,1}\d+$|$)/
```

##### 8.匹配--xxx=xxx的格式

```
.match(/--(\S*)=/)
```

##### 9.身份证校验

```
/^[1-9]\d{5}(18|19|([23]\d))\d{2}((0[1-9])|(10|11|12))(([0-2][1-9])|10|20|30|31)\d{3}[0-9Xx]$/
```

##### 10.银行卡号

```
/^([1-9]{1})(\d{15,18})$/
```

11.名称为中文、字母组合，长度为10

```
/^[\u4e00-\u9fa5_a-zA-Z]{1,10}$/
```
