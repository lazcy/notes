- #### Math常用的属性和方法

  - ###### 属性

  1. Math.E表示自然对数的底数或者称为基数，e约等于2.718，

     `Math.E = e 约等于 2.718`

  2. 圆周率

     `Math.PI 表示一个圆的周长与直径的比例，约为 3.14159：`

  3. 求数组最大值和最小值
  
     ```javascript
     var arr = [3,6,4];
     console.log(Math.max(arr)) // 输出6
     console.log(Math.min(arr)) // 输出3
     ```
  
  4. 去除数字的小数部分
  
     ```javascript
     Math.trunc = Math.trunc || function(x){
     	return x < 0 ? Math.ceil(x) : Math.floor(x);
     }
     ```
