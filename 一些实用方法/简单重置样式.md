##### 简单的重置样式

```css
*{
  margin: 0;
  padding: 0;
}
html,
body{
  position: relative;
  margin: 0 auto;
  /*max-width: 750px;*/
  background: #fff;
  font-family: "Microsoft soft";
}
img{
  width: 100%;
}
a:hover, a:visited, a:link, a:active {
  text-decoration: none;
  color: #333;
}
input{
  outline-style: none ;
  border: 0px solid #ccc; 
  background: rgba(255,255,255,0);
  padding: 0;
  width: 100%;
  font-size: 0.24rem;
}
```
