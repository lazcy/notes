#### 监听对象的变化

```javascript
watch:{
	obj:{
		handler(newVal, oldVal){
			console.log('obj change')
		},
		deep:true
	}
}
2、监听具体某个属性的变化
watch:{
	'obj.a':{
		handler(new, old){
			console.log(obj.a changed)
		}
	}
}
注：多层级的对象是监听不到的
如 obj:{
	a: {
		b: aa
	}
}
obj.a.b是监听不了属性的变化的
3、computed实现监听
computed:{
	a1(){
		return this.obj.a
	}
}
// 依赖改变时会重新计算新的值
```
